# 🌌 Python Astronomical Clock (全功能高精度天文时钟)

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Matplotlib](https://img.shields.io/badge/matplotlib-UI-orange)

这是一个基于 Python 和 `matplotlib` 构筑的多功能高精度天文时钟。它不仅能实时显示时间，还将**公历、农历、二十四节气、月相盈亏、实时潮汐、全球时区以及当地昼夜晨昏线**完美融合在一个极具极客感与艺术感的可视化表盘中。

## ✨ 核心特性

- **🕒 极客级时间显示**：高刷新的时/分/秒针联动，秒针附带流星动态拖影效果。
- **🌗 逼真天文物理模拟**：
  - **动态月相**：基于物理算法，利用椭圆模拟晨昏线，完美还原月亮每天的阴晴圆缺。
  - **潮汐追踪**：通过包络线与刻度动态展示由月相和时间共同作用的实时潮汐变化。
  - **昼夜光影**：内置 `astral` 库，根据所选城市的经纬度，精准计算日出日落时间，表盘背景会随当地真实昼夜状态呈现平滑渐变光影。
- **📅 双轨历法与节假日系统**：
  - **农历/公历双环**：外环显示公历进度，内环显示农历进度（已过日期标红，未过标绿）。
  - **智能节气与生肖**：自动高亮显示当前的十二生肖年份，精准标注二十四节气与中西法定节假日。
- **🖱️ 交互式查询系统**：
  - **拉环探索**：鼠标拖拽表盘上的蓝色（公历）或紫色（农历）微型圆环，即可实时查询全年中任意一天的具体日期与对应节日。
  - **一键切换时区**：侧边栏提供 24 个全球主要时区的快速切换，天文数据（日出日落、当地时间）会瞬间同步重算。
