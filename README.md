# 🌌 Python Astronomical Clock (全功能高精度天文时钟)

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Matplotlib](https://img.shields.io/badge/matplotlib-UI-orange)

这是一个基于 Python 和 `matplotlib` 构筑的多功能高精度天文时钟。它不仅能实时显示时间，还将**公历、农历、二十四节气、月相盈亏、实时潮汐、全球时区以及当地昼夜晨昏线**完美融合在一个极具极客感与艺术感的可视化表盘中。

## ✨ 核心特性

- **🕒 极客级时间显示**：高刷新的时/分/秒针联动，秒针附带流星动态拖影效果。
- **🌗 逼真天文物理模拟**：
  - **动态月相**：基于物理算法，利用椭圆模拟晨昏线，完美还原月亮每天的阴晴圆缺。
  - **潮汐追踪**：通过包络线与刻度动态展示由月相和时间共同作用的实时潮汐变化。
  - **昼夜光影**：内置 `astral` 库，根据所选城市的经纬度，精准计算日出日落时间，表盘背景会随当地真实昼夜状态呈现平滑渐变光影。
- **📅 双轨历法与节假日系统**：
  - **农历/公历双环**：外环显示公历进度，内环显示农历进度（已过日期标红，未过标绿）。
  - **智能节气与生肖**：自动高亮显示当前的十二生肖年份，精准标注二十四节气与中西法定节假日。
- **🖱️ 交互式查询系统**：
  - **拉环探索**：鼠标拖拽表盘上的蓝色（公历）或紫色（农历）微型圆环，即可实时查询全年中任意一天的具体日期与对应节日。
  - **一键切换时区**：侧边栏提供 24 个全球主要时区的快速切换，天文数据（日出日落、当地时间）会瞬间同步重算。

## 🛠️ 安装与运行

### 1. 环境依赖

运行本项目需要安装以下 Python 库：

```bash
pip install numpy matplotlib astral pytz lunardate
```

### 2. 运行程序
将代码克隆到本地后，直接运行主程序即可：

```bash
python main.py
```

### 3. 可选配置 (中心背景图)
程序支持在表盘中心显示自定义地球或星空背景图。您只需在同级目录下放置一张名为 world.jpg 的正方形图片即可。
程序在打包时（如使用 PyInstaller）也已做好了路径兼容处理。

- **🎮 交互指南**
  - **时区切换**：点击左侧面板的选项（如“UTC+8 北京”），时钟指针和昼夜光影会立即切换至当地状态。
  - **日期拉环**：
  - **鼠标左键**按住外侧蓝色小圈沿圆环拖动，右上角会浮现选中日期的公历信息及节假日。
  - **鼠标左键**按住内侧紫色小圈拖动，可查看农历详细信息。

- **🧩 代码结构简述**
  - **tz_data**: 全球各大城市经纬度与时区映射字典（用于太阳高度角计算）。
  - **get_sun_times()**: 利用 astral 精确计算对应经纬度的日出日落时间。
  - **draw_gradient_season()**: 绘制四季流转的色彩渐变环。
  - **on_press(), on_motion(), on_release()**: Matplotlib 的事件绑定，实现拉环的交互逻辑。
  - **update(frame)**: 核心动画渲染循环，以 50ms 的间隔驱动所有的天体数据刷新与 UI 重绘。
